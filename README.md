# Scam Honeypot Project

## Overview
This project simulates a phishing/scam interface to analyze user behavior and capture input data.

## Features
- Fake phishing interface
- Captures user input
- API integration for data handling
- Basic analysis of scam patterns

## Tech Stack
- HTML, CSS, JavaScript
- Python (API)

## How It Works
User interacts with a fake interface → data is captured → sent to backend API → stored/processed

## Purpose
To understand scam techniques and user interaction patterns for cybersecurity learning.

## Future Improvements
- Add dashboard for analysis
- Improve backend security
- Store data in database
